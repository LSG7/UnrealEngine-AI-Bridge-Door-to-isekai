1. This plugin is available for use for one month from its release date.  
   Once this period expires, please download and use the latest version.   
3. You must ask AI "execute how_to_use_this_tool mcp tool" first before starting any conversation with the AI.  
   This tool is a manual that provides the AI with instructions on how to use the plugin.
4. Ask AI to "explain using AI explainer."  
   This will generate a report on the explainer page.  
   The report can be explained with editor images and diagrams.  
   example : "Explain the blueprint graph to me using AI explainer."
5. The AI can identify actors or assets currently selected in the Editor. It can also detect nodes selected within the graph.  
6. Asking the AI to 'search for a solution online before proceeding' results in higher accuracy.  
   This reduces AI hallucinations and minimizes errors.  
   Note: This increases AI usage. Free tier users, please use with caution to avoid reaching your limits.  
7. Assets deleted via MCP tools can be recovered from 'Saved/Trash/'
8. Always use Git to track your repository. It allows you to revert your changes if something goes wrong.  
   Operations performed by AI can be reversed using Undo(Ctrl+z).  
   However, if the AI executes a large number of tasks, the Undo history may be filled quickly, making it difficult to revert to previous states.
